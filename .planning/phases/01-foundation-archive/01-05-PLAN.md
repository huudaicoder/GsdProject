**Application layer (Features/Departments/):**

1. **DTOs:**
   - `DepartmentDto.cs`: `record DepartmentDto(Guid Id, string Name, DateTime CreatedAt, DateTime? UpdatedAt)`
   - `CreateDepartmentRequest.cs`: `record CreateDepartmentRequest(string Name)`
   - `UpdateDepartmentRequest.cs`: `record UpdateDepartmentRequest(string Name)`

2. **Queries:**
   - `GetDepartmentsQuery.cs`: `record GetDepartmentsQuery(int Page, int PageSize, string? Search, string? SortBy, string? SortDir) : IRequest<PagedResult<DepartmentDto>>`
   - `GetDepartmentsQueryHandler.cs`: Follow the same pattern as `GetCategoriesQueryHandler`:
     - `context.Departments.AsNoTracking()`
     - Search filter: `.Where(d => d.Name.Contains(request.Search))` if search provided
     - Default sort by Name ascending
     - Pagination: Skip/Take
     - Project to DepartmentDto

3. **Commands:**
   - `CreateDepartmentCommand.cs`: `record CreateDepartmentCommand(string Name) : IRequest<DepartmentDto>`
   - `CreateDepartmentCommandValidator.cs`:
     - Name: NotEmpty with message "Ten phong ban khong duoc de trong.", MaxLength(100) with message "Ten phong ban khong duoc vuot qua 100 ky tu."
   - `CreateDepartmentCommandHandler.cs`: Create Department entity, add to context, save, return mapped DepartmentDto

   - `UpdateDepartmentCommand.cs`: `record UpdateDepartmentCommand(Guid Id, string Name) : IRequest<DepartmentDto>`
   - `UpdateDepartmentCommandValidator.cs`: Same rules as create
   - `UpdateDepartmentCommandHandler.cs`: Find by Id, throw NotFoundException if null, update Name, save, return DepartmentDto

   - `DeleteDepartmentCommand.cs`: `record DeleteDepartmentCommand(Guid Id) : IRequest<Unit>`
   - `DeleteDepartmentCommandHandler.cs`:
     - Find department by Id, throw NotFoundException if null
     - **Block delete check (D-22):** In Phase 1, Employee entity doesn't exist yet. Add placeholder check that returns 0 for now. `// TODO: Phase 2 — check Employee FK: var count = await context.Employees.CountAsync(e => e.DepartmentId == request.Id)`
     - When count > 0: throw `ConflictException("Khong the xoa phong ban nay — dang duoc su dung boi " + count + " nhan vien.")`
     - When count == 0: remove and save

4. **Controller (`DepartmentsController.cs`):**
   Follow the exact same pattern as `CategoriesController`:
   ```csharp
   [ApiController]
   [Route("api/[controller]")]
   [Authorize(Roles = "Admin")]
   public class DepartmentsController : ControllerBase
   {
       private readonly ISender _sender;
       public DepartmentsController(ISender sender) => _sender = sender;

       [HttpGet] // GET /api/departments?page=1&pageSize=20&search=&sortBy=name&sortDir=asc
       [HttpPost] // POST /api/departments
       [HttpPut("{id:guid}")] // PUT /api/departments/{id}
       [HttpDelete("{id:guid}")] // DELETE /api/departments/{id}
   }
   ```
   Each method sends the appropriate MediatR command/query via `_sender.Send(...)`.
