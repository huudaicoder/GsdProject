2. **AuthService** (`frontend/src/app/core/services/auth.service.ts`):
   - Injectable `providedIn: 'root'`
   - Private signal: `private _currentUser = signal<{ username: string; token: string } | null>(null)`
   - Public computed: `isAuthenticated = computed(() => this._currentUser() !== null)`
   - Public computed: `username = computed(() => this._currentUser()?.username ?? '')`
   - Constructor: check `localStorage.getItem('auth_token')` and `localStorage.getItem('auth_username')` — if both exist and token is not expired (decode JWT payload, check `exp` claim against `Date.now()/1000`), set `_currentUser` signal. This implements D-09 (persist across refresh).
   - `login(request: LoginRequest): Observable<LoginResponse>` — POST to `${environment.apiUrl}/auth/login`, on success: store `token` in `localStorage.setItem('auth_token', response.token)`, store `username` in `localStorage.setItem('auth_username', response.username)`, update `_currentUser` signal. Per D-08 (localStorage).
   - `logout()`: remove both localStorage items, set `_currentUser` to null, navigate to `/login`
   - `getToken(): string | null` — reads from `localStorage.getItem('auth_token')`

   - Functional interceptor using `HttpInterceptorFn` (Angular 19 standalone pattern)
